# AEO share-of-answer log

A DIY measure of how often PrepStep appears in AI answer engines for the queries
parents actually ask. Run the 10 prompts below once a month across the engines,
record whether PrepStep is mentioned (and how), and watch the trend as the SEO/AEO
surface, profiles, and reviews compound.

**Why this exists:** AI-referral traffic is tiny today but elite-quality and
growing. This log is the cheap, honest way to see whether the Loop 1 work is
moving the needle, and to catch misinformation or the EBSCO "PrepSTEP" name
collision surfacing in answers.

## How to run (≈10 minutes, monthly)
1. Open each engine fresh / logged-out where possible (avoids personalisation
   skew): **ChatGPT, Perplexity, Google AI Overview, Gemini, Copilot, Claude**.
2. Paste each of the 10 prompts.
3. For each engine × prompt, record in the month's table:
   - **Mentioned?** Y / N
   - **How** — rank in any list, sentiment, and crucially whether it correctly
     means prepstep.co.uk (UK 11+ app) vs EBSCO's "PrepSTEP" (collision flag).
   - Note which competitors were named (Atom, EdifyPod, PipPrep, Tassomai, Bond,
     CGP, 11plusapps) so we track the competitive set too.
4. Tally a simple **share-of-answer**: (engine×prompt cells mentioning PrepStep) /
   (total cells). Record the headline number + date below.

## The 10 prompts
1. What is the best 11+ exam prep app in the UK?
2. Best apps to prepare for the GL Assessment 11+ exam.
3. What 11+ practice apps are good for a Year 5 child?
4. What are some alternatives to Atom Learning for 11+ preparation?
5. Cheaper alternatives to Atom Learning for the 11 plus.
6. Is PrepStep a good app for the 11+? (brand)
7. PrepStep 11 plus review. (brand + collision test: does it return EBSCO PrepSTEP?)
8. Are there any free 11+ practice apps in the UK?
9. Best app for practising 11+ verbal reasoning.
10. Is there an 11+ prep app with an AI tutor?

## Baseline — 30 Jun 2026 (standard web search proxy; full engine run pending)
Captured via standard web search (approximates Google's index, not the chat
engines). This is the starting line; the first full 6-engine run replaces it.

- **"best 11+ app UK 2026"** → PrepStep **NOT mentioned**. Named instead:
  EdifyPod, Atom Learning, 11plusapps, PipPrep, Tassomai, Bond Online, CGP Online.
  Ranked listicles ([edifypod](https://edifypod.com/blog/best-11-plus-preparation-apps-uk-2026/),
  [11plusapps](https://www.11plusapps.com/blog/best-11-plus-apps.html)) omit PrepStep.
- **"PrepStep 11 plus review"** → returns EBSCO's **"PrepSTEP"** (US college/library
  test prep) and unrelated UK firms (PrepPlus, Prep4All). prepstep.co.uk **absent**.
  **Brand-name collision confirmed** — the disambiguating profiles + reviews target this.
- **Headline share-of-answer: ~0%.** PrepStep is effectively invisible to discovery
  and ambiguous on brand queries. Expected, the surface is days old. This is the
  number to beat.

### Competitive set observed (for reference)
- **PipPrep** (11plusapp.co.uk) — closest gamified rival (100k+ questions, boss
  battles, reward shop).
- **11plusapps.co.uk** — markets "8,190+ questions" (vs PrepStep's 8,210).
- **EdifyPod, Atom (£39.99/mo), Tassomai (£8.99/mo), Bond, CGP** — recurring names.
