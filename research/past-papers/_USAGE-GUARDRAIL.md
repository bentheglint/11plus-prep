# ⚠️ USAGE GUARDRAIL — read before using anything in this folder

The transcriptions in this folder are **verbatim copies of copyrighted CGP /
GL Assessment 11+ papers**, kept privately as a **benchmark reference only**.

## HARD RULE — never break

**Never copy a real-paper question into PrepStep's question bank — not
verbatim, not lightly reworded, not "same question with the numbers changed."**

When authoring PrepStep questions, you may mirror ONLY:
- the **question type / format** (e.g. "add-a-letter to make two words",
  "read a value off a graph then compute", running-passage cloze)
- the **difficulty mechanics** (number of reasoning steps, distractor
  temptingness, passage length & register)
- the **distractor-design principles** (e.g. distractors = the specific wrong
  methods a child actually computes)

You must NEVER reproduce:
- the actual question wording, numbers, word lists, codes, or answer options
- the reading passages (write entirely original passages)
- any recognisable near-clone of a specific real item

## Correct workflow
real paper → extract the *pattern* (type, difficulty, distractor logic,
format) → generate an **entirely original** question → verify it has no
verbatim/near-verbatim overlap with any source paper before it enters the bank.

## Distribution
- These files are git-tracked so the Oracle can benchmark across machines.
- They are **private reference**. Never surface, export, or ship real-paper
  content to users, marketing, or any public surface.
